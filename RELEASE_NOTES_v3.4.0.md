# Release Notes v3.4.0 - Social Features & Polish

## New Features
- **Social Network Integration**:
  - **Player Profiles**: View other players' profiles, stats, and highlights.
  - **Follow System**: Follow other players to see their activities.
  - **Activity Feed**: See updates from friends and the community.
  - **Connections Management**: Manage friends and friend requests.
  - **Discover**: Find new players based on skill level and location.

- **Enhanced User Experience**:
  - **Skeleton Loaders**: Improved loading states for a smoother experience.
  - **Pull-to-Refresh**: Added pull-to-refresh functionality for mobile users.
  - **Animations**: Added micro-animations for likes and interactions.

- **Analytics for Court Owners**:
  - **Revenue Charts**: Visual revenue tracking.
  - **Peak Hours Heatmap**: Identify busy and slow times.
  - **Smart Recommendations**: AI-powered suggestions for pricing and promotions.
  - **Customer Insights**: Deep dive into customer behavior.

## Improvements
- **Performance**: Optimized build size and loading times.
- **UI/UX**: Refined profile pages, highlight details, and navigation.
- **Type Safety**: Improved TypeScript coverage across the codebase.

## Bug Fixes
- Fixed issues with QR scanner integration.
- Resolved layout issues on mobile devices.
- Fixed navigation bugs in the social flow.
